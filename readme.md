#### 安装教程

1.把项目文件夹发到服务器根目录下，并启动服务器
2. 运行：location(服务器地址)/public

#### 使用说明

1. 导入数据库youbian.sql
2. 修改数据库配置：./env和./config/database.php
3. 基本用户:-u wonder -p wonde.123
4.后勤用户: -u utilities -p wonder.123
